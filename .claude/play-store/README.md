# Play Store listing — paste-ready content

Files in this folder match Play Console form fields. When verification clears,
open Play Console and paste each one into the matching section.

| File | Play Console section |
|---|---|
| `short-description.txt` | Store listing → Short description (80 char limit) |
| `full-description.md` | Store listing → Full description (4000 char limit) |
| `data-safety-answers.md` | App content → Data safety |
| `content-rating-answers.md` | App content → Content rating |
| `screenshot-checklist.md` | Store listing → Phone screenshots (capture on the Pixel) |
| `feature-graphic-spec.md` | Store listing → Feature graphic (1024×500 — needs design work) |

## Order to fill them out

1. Build + upload first AAB to internal testing (so the app exists in Play Console)
2. Capture screenshots on the Pixel (`screenshot-checklist.md`)
3. Design feature graphic (`feature-graphic-spec.md`)
4. Paste short + full description (`short-description.txt`, `full-description.md`)
5. Fill data safety form (`data-safety-answers.md`)
6. Complete content rating questionnaire (`content-rating-answers.md`)
7. App pricing & distribution → set as free, select countries
8. Review and submit
