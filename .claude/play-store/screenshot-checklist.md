# Screenshot capture checklist — Pixel

Capture in **portrait orientation**. Pixel's native screenshot resolution
(1080×2400 or similar) exceeds Play Store's minimum (320px shortest side, max
3840×3840), so any Pixel screenshot works.

Min 2 screenshots, max 8. **Recommend 6** — gives a complete tour without
overwhelming the listing.

## Capture sequence

### 1. Fretboard tab — the hero shot 🎯

- Open Fretboard tab
- Set root to **G**, scale to **Major**, mode **Scales**
- Position selector: tap **Pos 1** (so the highlighted position is visible)
- Note labels: **Note** (so dots show note names like "G", "B", "D")
- Take screenshot

This should show: scale notes lit up across the neck, position 1 highlighted
with the indigo gradient, the colored interval system (yellow root / red 3rd /
green 5th / blue 7th), the consolidated info panel at the bottom showing
Notes / Formula / Degrees in mono.

### 2. Chord Library — depth + theory

- Open Chords tab
- Pick **Cmaj7** from the drawer
- Take screenshot

Should show: large "C Major 7" title with "Seventh chord" eyebrow, framed
chord diagram with note labels, interval-structure cards (R / 3 / 5 / 7
with colored circles), "Often resolves to" carousel beneath.

### 3. Progressions — inspiration + audio

- Open Progressions tab
- Switch to **Examples** sub-tab
- Pick **"Dreamy descent"** (Shoegaze, Key of D major)
- Take screenshot

Shows: "Now playing" card with mono title, Dmaj7 displayed with intervals,
glowing accent play button, mini fretboard.

### 4. Practice — engagement / retention story

- Open Practice tab
- Tap **Name the Note → Beginner**
- Take screenshot mid-drill (after a few questions answered correctly so the
  score shows progress)

Shows: question prompt, fretboard with target note, note pills below, score
counter. This is the "daily-use" hook for the listing.

### 5. Tools — Metronome hero

- Open Tools tab → **Metronome**
- Set BPM to **120**, time signature **4/4**, hit play (so beat dots glow)
- Capture mid-beat (the lit dot should be visible)

Shows: large mono "120 BPM" hero card, glowing accent play button, beat dots,
feels like a piece of hardware.

### 6. Saved sheet — personalization

- From any tab, tap the **♥ Saved** button
- Have a few favorites already added (heart Cmaj7, Dreamy descent, G Major
  scale before this shot)
- Take screenshot

Shows: bottom sheet with Favorites tab, a few saved items with type-color
badges (CHORD / SCALE / PROG).

## Optional 7th — Tuning picker (if you want a 7th screenshot)

- Open Fretboard tab → tap **STD** pill in the TopBar
- Take screenshot of the modal showing the tuning list

## After capture

- Pixel saves to Photos → screenshots
- AirDrop / cloud-sync to your Mac, drop into a folder
- Upload via Play Console → Store listing → Phone screenshots
- Drag to reorder (Play Store shows them left-to-right; #1 has the most weight)
